Esse código é referente à um trabalho da matéria de inteligência artificial.
O objetivo era de implementar um algoritmo genético para resolver o seguinte problema:
6 palestrantes estão vindo de lugares diferentes do mundo com destino à Roma.
A equipe de organização do evento possui somente 1 van para busca-lós no aeroporto, dessa forma se torna necessário que os voos de chegada sejam proximos um dos outros, para que seja feita somente uma viagem do aeroporto para o hotel.
